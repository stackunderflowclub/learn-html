Header elements are semantic elements, meaning they have a special meaning in the markup. Headings are used for structuring, especially by screen readers, "reading mode" programs, and other document-parsing programs. Never use header tags to change text size! That can be accomplished later with styling using CSS (Cascading Style Sheets).

To render special characters which HTML parses, such as < and >, you can use escape sequences. They start with an & and end with a semicolon, with their abbreviation in between. For example, "<br />" was created with this: "&lg;br /&gt;" To escape the & and make it appear normally, you can use the escape sequence "&amp;"
